# 주석

1. 항상 한국어, 명령적 서술체로 작성할 것.
2. 함수나 클래스에 대한 주석을 작성할 때에는 최대한 대상의 추상적인 개념만을 설명할 것.
- 구체 구현에 대한 주석이 들어가는 순간, 나중에 내부 로직을 바꿨을 때 주석이 수정되는 것이 강요되지 않기 때문에
주석과 코드가 다른 구현이 될 수 있기 때문임.
